studio-customize plugin for `Grvlms <https://docs.grvlms.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/edutech-groove/grvlms-studio-customize

Usage
-----

::

    grvlms plugins enable studio-customize
